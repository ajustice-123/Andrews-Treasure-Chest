Supported Language(s): LabVIEW
Supported Model(s): WT310, WT310HC, WT332, WT333

Interface(s): GPIB, USB, Ethernet, Serial
Firmware Revision(s) Tested: F1.02

Certified: Yes

Source Code Available: Yes

Driver Revision: 1.0
Original Release Date: 02/22/2013
Current Revision Date: 02/22/2013

2. Required Software

Some software components need to be installed before using this instrument driver. The minimum versions of these components are listed below, and can be downloaded from the Download Site.

VISA 5.2 or later
LabVIEW 2010 or later

For more details, refer to the LabVIEW Help under Help » Search the LabVIEW Help.
